## 📋 О проекте

Платформа позволяет администратору создавать обучающие курсы, а сотрудникам — проходить их с отслеживанием прогресса.

### 🔥 Возможности

- **Конструктор курсов** — создавай разделы, уроки и тесты
- **PDF презентации** — загружай и смотри прямо в браузере
- **Полноэкранный режим** — удобный просмотр материалов
- **Тесты** — вопросы с выбором ответа или текстовым вводом
- **Прогресс** — отслеживание пройденных уроков
- **Роли** — администратор и сотрудник
